# <h1 align="center"> bn.rs </h1>

**[bn.js](https://github.com/indutny/bn.js/) bindings for Rust & WebAssembly**

## Install
Install dependencies
```console
yarn
```
Build Chrome Extension
```console
yarn build
```
Then add extension to Chrome from `dist` folder
